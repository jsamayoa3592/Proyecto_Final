# Proyecto Final - Cotizador de Seguro

## Descripción
Este programa calcula la cotización de un seguro tomando un precio base y agregando recargos según los datos que ingrese el usuario.

## Funcionamiento
El programa primero solicita el nombre y la edad del asegurado.
Si la persona es menor de 18 años, no se puede realizar la cotización.

Si es mayor de edad, se pregunta si está casado y si tiene hijos.
Dependiendo de las respuestas, se aplican los siguientes recargos:

- Según la edad del asegurado.
- Según la edad del cónyuge (si está casado).
- 20% del precio base por cada hijo.

Al final se suman todos los recargos al precio base y se muestra el precio final.

## Posibles mejoras
- Mejorar la validación de datos.
- Permitir realizar varias cotizaciones sin reiniciar el programa.
- Mostrar el detalle de cada recargo por separado.
